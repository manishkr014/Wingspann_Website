# Wingspann Website

This is the official repository for the state-of-the-art Wingspann corporate website. The website is built using React, Vite, and Tailwind CSS. It features a modern, dynamic, and responsive design tailored for a leading company in the aerospace and autonomous systems industry.

## Features
- **Responsive Navigation**: Full mobile support with a sleek hamburger menu.
- **Dynamic Headers**: High-performance scrolling animations, fading elements, and fluid gradients.
- **Intersection Animations**: Utilizing Framer Motion for smooth scroll-into-view slide and fade effects.
- **Video Backgrounds**: Engaging abstract AI-generated video hero sections.
- **Modern UI Styling**: High contrast black, white, and red color scheme in theme with aerospace innovation.

## Technology Stack
- **Framework**: React 18+
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, Vanilla CSS (`index.css` for custom animations)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router DOM (v6)

## Setup and Installation

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation Steps
1. Clone the repository
   ```bash
   git clone https://github.com/manishkr014/Wingspann_Website.git
   ```
2. Navigate to the project directory
   ```bash
   cd "Wingspann Website 230226/project"
   ```
3. Install the dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```

## Production Build
To create a production-ready build, run:
```bash
npm run build
```
This will generate optimized files in the `dist` folder.
